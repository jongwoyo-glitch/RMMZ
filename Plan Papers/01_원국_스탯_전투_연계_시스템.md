> v0.1 | 2026-04-18 | 큰 틀 설계 (구현 전 개념 단계)

## 0. 설계 철학

- **클래스 없음** — 캐릭터의 정체성은 타고난 원국 + 쌓아온 스킬 성향의 조합으로 결정
- **잡캐 가능, 파티 유도** — 혼자 다 할 수 있지만 오행 상생/상극이 파티 시너지를 자연스럽게 만듦
- **전투 밖에서도 의미** — 같은 오행이라도 전투/생활에서 발현이 다름
- **시각적 정체성** — Warhammer Total War식 레벨 숫자 + 오행 색상 파이 차트

## 1. 오행 (五行) — 시스템의 근간

### 1-1. 다섯 원소

| 오행 | 한자 | 색상 | 상생 (나를 낳는) | 상극 (내가 이기는) |
|------|------|------|------------------|-------------------|
| 목(木) | Wood | 초록 (#44bb44) | 수 → 목 | 목 → 토 |
| 화(火) | Fire | 빨강 (#dd4444) | 목 → 화 | 화 → 금 |
| 토(土) | Earth | 노랑 (#ccaa44) | 화 → 토 | 토 → 수 |
| 금(金) | Metal | 흰색 (#ccccdd) | 토 → 금 | 금 → 목 |
| 수(水) | Water | 파랑 (#4488cc) | 금 → 수 | 수 → 화 |

### 1-2. 상생/상극 관계

```
상생 순환 (생 → 생):   목 → 화 → 토 → 금 → 수 → 목
상극 순환 (극 → 극):   목 → 토 → 수 → 화 → 금 → 목
```

- **상생**: 시너지/보너스. 파티에 목+화 조합이면 화 측 버프 등
- **상극**: 카운터/억제. 목 성향이 강한 캐릭터가 토 성향 적에게 유리 등

### 1-3. 오행 ≠ 속성 데미지 타입

오행은 "데미지 속성"이 아니라 **"전투 성향"**이다.
같은 목(木)이라도 스킬 카테고리에 따라 완전히 다르게 발현된다:

| 오행 | 물리-근접 | 물리-원격 | 마법-공격 | 마법-보조 |
|------|-----------|-----------|-----------|-----------|
| 목 | 속공·돌진, 많은 타수, 단일대상 | 직사화기, 연발 사격 | 번개: 직선 관통, 다단히트 | 소환: 정령 소환, 소환수 강화 |
| 화 | 양손·대형, 범위공격, 큰 데미지 | 직사화기, 범위·중거리 사격 | 화염: 누킹, 방사형, 직관적 | 회복: 체력회복, 망자회생 |
| 토 | 한손+방패, 탱커, 버프, 협동공격 | 손쇠뇌·쌍수, 터렛·엄폐물 설치 | 대지: 좁은 범위 CC, 도트뎀 | 전투버프: 공방 버프, 배리어, 지형변형 |
| 금 | 양손·폴암, 치명타, 카운터, 중거리 | 곡사+직사, 초장거리, 고크리뎀 | 바람: 넓은 범위, 빠른속도, 크리 위주 | 카운터: 채널링 끊기, 투사체 반격, 디스펠 |
| 수 | 한손·단검, 은신, 보조행동 변수창출 | 한손 손쇠뇌, 속성볼트, 함정 | 물/얼음: 행동력 저하, CC, 중첩뎀 | 디버프: 상태이상 장판, 강제이동, 혼란·매혹 |

## 2. 원국 (原局) — 캐릭터의 타고난 기질

### 2-1. 사주팔자 구조

전통 사주팔자에서 4개의 기둥(柱), 각 기둥에 천간(天干)+지지(地支) = 8글자:

```
        년주    월주    일주    시주
천간:   년간    월간    일간    시간
지지:   년지    월지    일지    시지
```

- **천간 10종**: 갑을병정무기경신임계 (甲乙丙丁戊己庚辛壬癸)
  - 양목, 음목, 양화, 음화, 양토, 음토, 양금, 음금, 양수, 음수
- **지지 12종**: 자축인묘진사오미신유술해 (子丑寅卯辰巳午未申酉戌亥)
  - 각각 오행+음양 속성 보유, 일부는 복합 오행(지장간)

### 2-2. 게임 내 활성화 규칙

| 캐릭터 유형 | 활성 글자 | 결정 방식 |
|-------------|-----------|-----------|
| **플레이어** | 일간 + 일지 (2글자) 선택, 나머지 6글자 = 성장 변동 | 캐릭터 생성 시 선택 |
| **동료 캐릭터** | 8글자 전부 활성 | 사전 설정 (고정) |
| **엑스트라 NPC** | 2글자만 활성 | 사전 설정 (간이) |

### 2-3. 원국이 게임에 미치는 영향

```
원국 8글자
  ├─ 초기 스탯 성향 (어떤 오행 스탯이 높게 시작하는가)
  ├─ 스킬 학습 적성 (상생 오행 스킬은 저렴, 상극 오행 스킬은 고비용)
  ├─ 비전투 성향 (NPC 호감도 보정, 생활스킬 적성)
  └─ 대운/세운 보정 (게임 내 시간 흐름에 따른 버프/디버프 변동) ← 향후 확장
```

### 2-4. 플레이어 성장에 따른 원국 변동

- 일간·일지(선택된 2글자)는 **불변** — 캐릭터의 근본
- 나머지 6글자는 성장 선택에 의해 채워짐:
  - 스킬을 찍을 때마다 해당 오행이 빈 칸에 반영
  - 특정 이벤트/선택지에서 글자가 바뀔 수 있음
  - 8글자가 모두 채워지면 → 원국 완성 → 고유 칭호/특수 효과 등

## 3. 스킬 체계

### 3-1. 대분류

```
스킬
├─ 전투 스킬
│   ├─ 물리-근접
│   ├─ 물리-원격
│   ├─ 마법-공격
│   └─ 마법-보조
│
└─ 생활 스킬
    ├─ 자원/제작 계열 (채광, 약초, 대장장이, 연금술 등)
    └─ 사회/교류 계열 (상술, 외교, 설득, 정보수집 등)
```

### 3-2. 전투 스킬 × 오행 매트릭스

각 전투 스킬은 **카테고리(4종) × 오행(5종) = 20가지 성향 슬롯**에 배치된다.

```
         목(木)   화(火)   토(土)   금(金)   수(水)
물리근접  [    ]   [    ]   [    ]   [    ]   [    ]
물리원격  [    ]   [    ]   [    ]   [    ]   [    ]
마법공격  [    ]   [    ]   [    ]   [    ]   [    ]
마법보조  [    ]   [    ]   [    ]   [    ]   [    ]
```

각 슬롯 안에 여러 개의 구체적 스킬이 들어간다.
예시: `물리근접-목 = [연속베기, 돌진, 뒤잡기, ...]`

### 3-3. 스킬 학습과 오행 적성

```
학습 비용 계산:
  기본 비용 × 적성 보정

적성 보정:
  원국에 해당 오행이 많으면     → 비용 ↓ (적성 높음)
  원국에 상생 오행이 많으면     → 비용 약간 ↓
  원국에 상극 오행이 많으면     → 비용 ↑ (적성 낮음)
  원국에 해당 오행이 전혀 없으면 → 비용 ↑↑ (비적성)
```

잡캐로 키우는 것은 가능하지만, 비용이 더 든다.

### 3-4. 생활 스킬

- 전투 스킬과 동일하게 오행 성향을 가짐
- 하지만 전투 판정에 직접 관여하지 않음
- 원국의 비전투 면(일주 외의 글자들)이 생활 스킬 적성에 영향

## 4. 레벨과 시각 표현

### 4-1. 레벨 = 총 스킬 포인트 투자량

- 전투/생활 스킬 투자 합산이 레벨
- 특정 레벨 구간마다 스탯 상승, 새 스킬 슬롯 해금 등

### 4-2. 파이 차트 정체성 (Warhammer TW 스타일)

```
┌──────────────────┐
│   Lv. 24         │
│   ┌───────┐      │
│   │  ◐    │      │  ← 5색 원형 파이
│   │ 오행  │      │     투자 비율에 따라 색상 면적 변동
│   │ 파이  │      │
│   └───────┘      │
│   김철수          │
└──────────────────┘
```

- 원형 파이의 각 섹터 = 투자한 오행별 비율
- 특정 오행이 50% 이상이면 "주 성향" 표시 (테두리 글로우 등)
- 균등 분배(20% × 5)면 "무극(無極)" 같은 특수 칭호 가능

## 5. 전투 밸런싱 관여 축 (확장 후)

### 현재 (기존 SRPG_Core)
| 축 | 요소 |
|----|------|
| 수치 | ATK, DEF, AGI, HP, MP, LUK |
| 위치 | MOV, 사거리, 지형차단, 엄폐 |
| 템포 | AP 턴 시스템, 행동 경제(주+보조) |

### 원국 시스템 도입 후 (추가되는 축)
| 축 | 요소 |
|----|------|
| 성향 | 오행 적성에 따른 스킬 효율 차등 |
| 시너지 | 상생 파티 버프, 상극 카운터 |
| 상태 | 오행 기반 버프/디버프 (향후) |
| 적성 | 스킬 학습 비용 차등 → 빌드 다양성 |
| 성장 | 원국 변동 → 중후반 특화 vs 범용 선택 |
