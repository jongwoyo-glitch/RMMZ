### 오행 순환

```
상생(生) — 강화 관계:

  "번개가 불을 일으키고, 불이 재(흙)를 만들고, 흙이 금맥을 품어 바람을 세우고,

상극(克) — 제압 관계:
  木(천둥) → 土(흙) → 水(물/얼음) → 火(불) → 金(바람) → 木(천둥)
  "번개가 대지를 가르고, 흙이 물을 막고, 물이 불을 끄고,
   불이 바람을 삼키고, 바람이 뇌운을 흩뜨린다"
```

## A2. 상생(生) — 상위 연계 규칙

상생은 데미지 퍼센테이지 보너스가 아니라, **상위 상태이상 또는 상위 장판으로 승급**하는 방식.
선행 원소가 후행 원소와 만나면 → **새로운 파생 상태/장판이 탄생**.

### A2.1 스킬 → 유닛 상태 (상태 승급)

유닛이 선행 원소의 상태를 갖고 있을 때 후행 원소 공격을 받으면, 기존 상태가 **상위 상태로 변환**.

| 유닛 상태 (선행) | 공격 속성 (후행) | 결과 상태 | 효과 |
|-----------------|-----------------|-----------|------|
| 대전(木) | 불(火) 공격 | **뇌염(雷炎)** | 턴당 HP -15%, 인접 1칸 화상 전이, 2턴 |
| 화상(火) | 흙(土) 공격 | **소토(燒土)** | DEF -25%, 이동 -2, 지면 고착 2턴 |
| 접지(土) | 바람(金) 공격 | **사폭(砂暴)** | 실명(HIT -50%), 이동 -3, 2턴 |
| 풍압(金) | 물/얼음(水) 공격 | **한파(寒波)** | 이동 -3, 공속 -25%, EVA -20%, 2턴 |
| 젖음(水) | 천둥(木) 공격 | **감전(感電)** | 1턴 행동불가 + 인접 유닛 번개 스플래시 |

> 핵심: 기존 기본 상태(대전·화상 등)가 **소모되어 사라지고**, 대신 더 강력한 파생 상태가 부여됨.
> 특히 **水生木 = 젖음 + 천둥 = 감전**은 매우 직관적인 상호작용.

### A2.2 장판 → 스킬 (장판 승급)

선행 원소 장판 위에 후행 원소 스킬이 착탄하면, 장판이 **상위 장판으로 변환**.

| 장판 (선행) | 스킬 (후행) | 결과 장판 | 효과 |
|------------|------------|-----------|------|
| 화염(火) 장판 | 土 스킬 | **용암 지대** | 이동 -3, DEF -20%, 화상+접지 동시 |
| 진흙(土) 장판 | 金(바람) 스킬 | **사폭 지대** | 모래폭풍, 실명+이동↓ |
| 돌풍(金) 장판 | 水 스킬 | **한파 지대** | 차가운 폭풍, 이동 -3, 젖음+풍압 동시 |

> **木生火 장판 승급**: 木(천둥)은 매질 없이 단독 장판이 될 수 없으므로,
> 木→火 상생의 장판 승급은 木계열 파생 장판(감전수 등)에 火가 닿을 때 발생.
>
> **水生木 장판 승급**: 수면(水) + 천둥(木) 스킬 → **감전수** (매질 결합, A4 참조).

> 핵심: 장판이 단순히 강화되는 게 아니라 **새로운 복합 장판으로 변환**됨. 원본 장판의 턴은 리셋.

## A3. 상극(克) — 상호작용 생성 규칙

상극은 단순 제거가 아니라, 극하는 과정에서 **새로운 부산물(상태/장판)이 발생**.

### A3.1 스킬 → 유닛 상태 (상태 반응)

| 유닛 상태 (피극) | 공격 속성 (극하는) | 결과 | 부산물 |
|-----------------|-------------------|------|--------|
| 접지(土) | 천둥(木) 공격 | 접지 해제 | **낙뢰(落雷)** — 접지가 번개를 끌어당겨 관통, 고정 피해 + 1턴 스턴 |
| 젖음(水) | 흙(土) 공격 | 젖음 해제 | **진흙 속박** — 이동 -4, 2턴 |
| 화상(火) | 물/얼음(水) 공격 | 화상 해제 | **증기** — 해당 타일+인접 1칸 실명 1턴, 시야 차단 |
| 풍압(金) | 불(火) 공격 | 풍압 해제 | **열풍(熱風)** — 뜨거운 바람 확산, 인접 1칸 화상 전이 |
| 대전(木) | 바람(金) 공격 | 대전 해제 | **방전(放電)** — 바람이 전하를 흩뜨림, 주변 2칸 약 대전 확산 |

> 핵심: "끄는" 행위 자체가 **새로운 전장 효과**를 만든다. 물로 불을 끄면 연기, 바람으로 번개를 흩뜨리면 방전.

### A3.2 스킬 → 장판 (장판 반응)

극하는 원소 스킬이 장판에 닿으면, 장판이 소멸하면서 **잔류 효과 장판이 남음**.

| 장판 (피극) | 스킬 (극하는) | 장판 소멸 | 잔류 효과 |
|------------|--------------|-----------|-----------|
| 진흙(土) 장판 | 木(천둥) 스킬 | 土 소멸 | **낙뢰 지대** — 1턴 스턴 장판 |
| 수면(水) 장판 | 土 스킬 | 水 소멸 | **진흙 수렁** — 속박(이동 -4) 장판으로 변환 |
| 화염(火) 장판 | 水 스킬 | 火 소멸 | **증기 구름** — 실명+시야차단 장판, 2턴 |
| 돌풍(金) 장판 | 火 스킬 | 金 소멸 | **열풍 잔류** — 인접 화상 전이, 1턴 |
| 木계열 파생 장판* | 金(바람) 스킬 | 파생 소멸 | **방전 잔류** — 약 대전 확산, 1턴 |

> *木계열 파생 장판: 감전수(매질결합) 등. 木(천둥)은 단독 장판이 없으므로 파생만 존재.

### A3.3 장판 × 장판 (장판끼리 충돌)

같은 타일에 극하는/극받는 원소 장판이 만나면:

| 기존 장판 | 새 장판 | 결과 |
|-----------|---------|------|
| 水 | 土 | **진흙 수렁** (합성 장판), 양쪽 소멸 |
| 火 | 水 | **증기 구름**, 양쪽 소멸 |
| 金 | 火 | 金 소멸, **火 장판 유지** + 열풍 잔류 |
| 土 | 木(파생) | 土 소멸 + 낙뢰 잔류 |

> 木(천둥)은 단독 기본 장판이 없으므로 木 vs X 기본 장판 충돌은 발생하지 않음.
> 木계열 파생 장판(감전수 등)이 다른 장판과 충돌 시, 파생 장판의 주 속성(木)에 따라 상극 판정.

## A4. 동일 오행 (비화/비겁)

같은 오행끼리 만나면 — **효과 갱신 + 턴 갱신만** (강화 없음).

| 상황 | 효과 |
|------|------|
| 같은 원소 장판 중첩 | 지속시간 리셋, 효과 동일 유지 (강화 안 됨) |
| 같은 원소 상태 재부여 | 지속턴 리셋만 |

### 水 내부: 물 vs 얼음

물(5)과 얼음(3)은 같은 水이지만 형태가 다름.

| 상황 | 효과 |
|------|------|
| 물 장판 + 얼음 스킬 | 물 → **빙판** (미끄러짐, 이동 페널티) |
| 얼음 장판 + 火 스킬 | 얼음 → **물 장판** (해빙, 상극 아닌 형태 변환) |
| 얼음 장판 + 기타 | **추가 상호작용 없음** — 얼음은 녹을 수만 있음 |
| 물 장판 + 물 스킬 | 지속시간 리셋만 |

> 핵심: 얼음은 단순(녹는 것만). 물은 다양한 매개체 역할 (전기 전도, 빙결 등).

### 木(천둥)의 매질 결합

**木(천둥)은 매질 없이 단독 장판으로 존재할 수 없다.** 전격은 항상 전도체(물, 금속, 흙)를 통해서만 장판화.

| 매질 장판 | + 木(천둥) 스킬 | 결과 | 효과 |
|-----------|----------------|------|------|
| 수면(水) 장판 | 천둥 스킬 | **감전수** | 물 장판 **전체 범위**에 대전 부여 → 감전수 장판 변환 (水生木 매질결합) |
| 진흙(土) 장판 | 천둥 스킬 | **뇌진(雷陣)** | 번개가 흙을 관통, 낙뢰 지대로 변환 (木克土 상극 반응) |
| 기름 웅덩이 | 천둥 스킬 | **전도 폭발** | 즉시 HP -15%, 대전 부여, 기름 소멸 |

> 木 스킬이 매질 없는 빈 타일에 착탄하면: 유닛에 대전 상태만 부여, 장판 생성 안 됨.
> 木계열 파생 장판(감전수 등)은 木 속성을 가지므로 金克木 상극의 대상이 됨.

## A5. 오행 외 속성 (물리, 빛, 어둠)

오행 순환에 참여하지 않지만, 장판과 상호작용하는 독자 규칙:

| 속성 | 장판/상태 상호작용 |
|------|-------------------|
| 물리(1) | 장판과 상호작용 없음. 순수 물리 피해 |
| 빛(8) | 어둠 장판 **정화** (제거). 어둠 상태 해제. 저주 장판 중화 |
| 어둠(9) | 빛 장판 **부패** (제거). 축복 상태 해제. 치유 장판 → 저주 변환 |

빛/어둠은 **대립쌍**으로 서로를 상쇄하는 별도 축.

## A6. 유닛 원소 부여 + 연쇄반응

장판 위 유닛은 해당 장판의 원소 상태를 부여받는다.
이 상태를 갖고 있는 유닛에 다른 원소 공격이 닿으면, **유닛 상태 반응 + 장판 반응이 동시에 발생**한다.

### 연쇄반응 예시

```
시나리오 1: 화염 장판 위에서 화상 상태인 유닛 A에 물 스킬 착탄

1단계 — 유닛 상태 반응 (A3.1 상극: 水克火):
  화상(火) + 水 공격 → 화상 해제 + 증기(실명 1턴) 발생

2단계 — 장판 반응 (A3.2 상극):
  물 스킬의 AoE 범위 내 화염 타일에 대해:
  화염(火) 장판 + 水 스킬 → 화염 타일 소멸 + 증기 구름 잔류

결과:
  - 유닛 A: 화상 해제, 실명 1턴
  - 물 스킬 범위 내 화염 타일: 증기 구름으로 변환
  - 범위 밖 화염 타일: 그대로 유지

시나리오 2: 수면 장판 위에서 젖음 상태인 유닛 B에 천둥(木) 스킬 착탄

1단계 — 유닛 상태 반응 (A2.1 상생: 水生木):
  젖음(水) + 천둥(木) 공격 → 젖음 소모 → **감전(感電)** 승급! 1턴 행동불가

2단계 — 장판 반응 (A4 매질 결합):
  수면(水) 장판 + 천둥(木) 스킬 → 물 장판 전체가 **감전수**로 변환

결과:
  - 유닛 B: 감전 상태 (행동불가 + 인접 스플래시)
  - 수면 장판 전체: 감전수 장판으로 승급
```

### 처리 순서

```
스킬 착탄 시 처리 플로우:

1. 타겟 유닛의 기존 원소 상태 체크
   → 상생이면 A2.1 (상태 승급)
   → 상극이면 A3.1 (상태 반응 + 부산물)

2. 스킬 AoE 범위 내 장판 타일 체크 (타일별 개별 처리)
   → 상생이면 A2.2 (장판 승급)
   → 상극이면 A3.2 (장판 소멸 + 잔류)
   → 매질 결합이면 A4 (木 전용)
   → 동일이면 A4 (갱신만)

3. 새 장판 생성 스킬이면 → 기존 장판과 A3.3 충돌 체크

4. 부산물 상태/장판 적용 (증기, 진흙 등)
```

> 핵심: **유닛 반응과 장판 반응은 독립적으로 각각 발생**. 하나의 스킬이 두 계층 모두 트리거.

# Part B. 장판 시스템

## B1. 장판 그래픽 — 타일별 스프라이트 방식

장판은 단일 이미지가 아니라, **그리드 타일 하나당 개별 스프라이트**로 구성.

```
장판 렌더링 구조:

┌───┬───┬───┐
│ S │ S │ S │   S = 개별 스프라이트 (루프 애니메이션)
├───┼───┼───┤       - 각 타일이 독립적
│ S │ S │ S │       - 개별 소멸/변환 가능
├───┼───┼───┤       - 그리드 1칸보다 살짝 큰 크기
│ S │ S │ S │       - 유닛 초상화 뒤(아래 레이어)에 배치
└───┴───┴───┘
```

### 타일 스프라이트 규격

| 항목 | 값 |
|------|-----|
| 크기 | 그리드 1칸 × ~1.1 (약간 오버랩하여 바둑판 경계 매끄럽게) |
| 레이어 | 유닛 스프라이트 뒤, 지형 위 |
| 애니메이션 | 프레임 루프 (불꽃 일렁임, 물결, 바람 소용돌이 등) |
| 제거 | **타일별 개별 소멸** — 부분 소거 가능 (상극 스킬 AoE로 일부 타일만 제거) |
| 투명도 | 잔여 턴에 따라 페이드아웃 |

### 장판 데이터 구조 (타일 배열 방식)

```javascript
// SrpgField 객체
{
    id: "field_001",
    element: "fire",        // 오행 키 또는 "none"
    tiles: [                // 점유 타일 배열 — 개별 관리
        { x: 5, y: 3, sprite: Sprite_FieldTile },
        { x: 5, y: 4, sprite: Sprite_FieldTile },
        { x: 6, y: 3, sprite: Sprite_FieldTile },
    ],
    stateId: 32,            // 부여할 상태이상
    duration: 3,            // 잔여 턴
    casterId: 7,            // 시전자 유닛 ID (데미지 귀속)
    ap: 0,                  // 턴 오더 참여용 AP
    agi: 300,               // 장판 AGI (턴 오더 속도)
}
```

## B3. 비원소 장판

오행과 무관. 순수하게 상태이상만 부여하는 기술 기반 장판.
원소 상호작용에 참여하지 않으므로, 다른 원소 장판/스킬로 제거되지 않음.
동일 위치에 원소 장판과 공존 가능 (각자 독립 적용).

**지속 턴은 스킬별 개별 설정** — 전역 기본값 없음. 각 스킬의 노트태그에서 정의.

| 장판명 | 상태 부여 | 효과 | 생성 수단 |
|--------|-----------|------|-----------|
| 매혹의 안개 | 매료(9) | 적/아 구분 혼란 | 마법/아이템 |
| 공포의 영역 | 공포(48) | 장판 방향으로 이동 불가 | 마법 |
| 혼란의 장막 | 혼란(8) | 랜덤 행동 | 마법 |
| 침묵의 결계 | 침묵(6) | 스킬 사용 불가 | 마법/아이템 |
| 수면의 안개 | 수면(10) | 행동불가 (피격 시 해제) | 마법 |
| 치유의 샘 | HP재생(15) | 턴당 HP 회복 | 마법 |
| 결계 | 방어(2) | 피해 감소 | 마법 |
| 분노의 전장 | 분노(7) | ATK↑, DEF↓ | 마법 |
| 독안개 | 독(4) | 턴당 HP 감소 | 스킬/아이템 |
| 기름 웅덩이 | 유침(49) | 이동↓, 화염 취약 +100% | 아이템 |

### 비원소 장판 노트태그 예시

```
<srpgField>
type: none
state: 48
duration: 3
range: diamond 2
animation: fear_fog
</srpgField>
```

### 비원소 장판의 특수 규칙

- 비원소 장판끼리 중첩: **후발 우선** (동일 상태면 갱신, 다르면 대체)
- 비원소 + 원소 장판: **공존** 가능 (같은 타일에 독립 존재)
- 예외 — **독안개, 기름**: 오행은 아니지만 **火와 반응** (연소/폭발)

```
독안개 + 火 스킬/장판 → 폭발 (즉시 HP -25%) + 독안개 소멸 + 화염 잔류
기름 + 火 스킬/장판 → 대화재 변환 (기름 소멸, 화염 장판 강화)
기름 + 木(천둥) 스킬 → 전도 폭발 (즉시 HP -15%) + 기름 소멸 + 대전 부여
```

# Part C. 상호작용 처리 타이밍

## C1. 발생 시점

```
1. 스킬 착탄 시
   a) 타겟 유닛의 기존 상태 체크 → 상생 승급 / 상극 부산물
   b) 스킬 AoE 범위 내 장판 타일 개별 체크 → 장판 승급 / 소멸+잔류
   c) 새 장판 생성 스킬이면 → 기존 장판과 충돌 체크 (타일별)

2. 장판 위로 유닛 이동 완료 시
   → 장판 상태 부여

3. 장판의 턴 (AP 도래)
   → 위의 유닛에 상태 갱신 (turns 리프레시)
   → duration 감소 (타일 전체 동시)

4. 유닛이 장판 밖으로 이탈 시
   → 상태의 자체 턴 카운팅 시작
```

## C2. 처리 우선순위

같은 타이밍에 여러 반응이 가능할 때:

```
1. 상극 > 상생 (극하는 관계가 먼저)
2. 유닛 상태 반응 → 장판 반응 (유닛 먼저, 장판 두 번째)
3. 장판 반응은 타일별 개별 처리 (AoE 범위 내 타일만 영향)
4. 부산물(증기 등) 생성 → 부산물의 상태 적용
```
